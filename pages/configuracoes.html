🔌 API de Integração — Firestore REST (gratuita)

Envie colaboradores de qualquer sistema externo diretamente para o Firestore usando a REST API nativa do Firebase — 100% gratuita, sem necessidade de Firebase Functions.

Como funciona: O sistema externo faz login anônimo no Firebase para obter um token, depois usa esse token para escrever diretamente no Firestore via HTTPS.
📖 Documentação completa da API

🔑 PASSO 1 — Obter token de acesso

Método: POST — válido por 1 hora, renove quando expirar

https://identitytoolkit.googleapis.com/v1/accounts:signInAnonymously?key=AIzaSyD7E3w1x9ezsU_59brfx75mJ6V2xwDpID4

Body: {}  |  Resposta: {"idToken": "eyJ..."}

👤 PASSO 2 — Criar/Atualizar colaborador

Método: PATCH — usa matrícula como ID (MAT_00123) para evitar duplicatas

https://firestore.googleapis.com/v1/projects/gestao-premios-indicacao/databases/(default)/documents/colaboradores/MAT_{MATRICULA}

Header: Authorization: Bearer {idToken}

🐍 Exemplo completo em Python

import requests

API_KEY = "AIzaSyD7E3w1x9ezsU_59brfx75mJ6V2xwDpID4"
PROJECT = "gestao-premios-indicacao"
BASE = f"https://firestore.googleapis.com/v1/projects/{PROJECT}/databases/(default)/documents"

def get_token():
    r = requests.post(
        f"https://identitytoolkit.googleapis.com/v1/accounts:signInAnonymously?key={API_KEY}",
        json={}
    )
    return r.json()["idToken"]

def upsert_colaborador(token, mat, nome, setor, data_adm, tipo="experiencia"):
    url = f"{BASE}/colaboradores/MAT_{mat}"
    body = {"fields": {
        "matricula":    {"stringValue": mat},
        "nome":         {"stringValue": nome},
        "setor":        {"stringValue": setor},
        "dataAdmissao": {"stringValue": data_adm},
        "tipoContrato": {"stringValue": tipo},
        "status":       {"stringValue": "ativo"},
        "foiIndicado":  {"booleanValue": False},
        "fonte":        {"stringValue": "api"}
    }}
    r = requests.patch(url, json=body, headers={"Authorization": f"Bearer {token}"})
    print(r.status_code, r.json())

token = get_token()
upsert_colaborador(token, "00123", "João Silva", "Produção", "2024-03-01")

🟨 Exemplo completo em JavaScript / Node.js

const API_KEY = "AIzaSyD7E3w1x9ezsU_59brfx75mJ6V2xwDpID4";
const PROJECT = "gestao-premios-indicacao";
const BASE = `https://firestore.googleapis.com/v1/projects/${PROJECT}/databases/(default)/documents`;

async function getToken() {
  const r = await fetch(
    `https://identitytoolkit.googleapis.com/v1/accounts:signInAnonymously?key=${API_KEY}`,
    { method: "POST", body: "{}", headers: { "Content-Type": "application/json" } }
  );
  return (await r.json()).idToken;
}

async function upsertColaborador(token, mat, nome, setor, dataAdm, tipo = "experiencia") {
  await fetch(`${BASE}/colaboradores/MAT_${mat}`, {
    method: "PATCH",
    headers: { Authorization: `Bearer ${token}`, "Content-Type": "application/json" },
    body: JSON.stringify({ fields: {
      matricula:    { stringValue: mat },
      nome:         { stringValue: nome },
      setor:        { stringValue: setor },
      dataAdmissao: { stringValue: dataAdm },
      tipoContrato: { stringValue: tipo },
      status:       { stringValue: "ativo" },
      foiIndicado:  { booleanValue: false },
      fonte:        { stringValue: "api" }
    }})
  });
}

const token = await getToken();
await upsertColaborador(token, "00123", "João Silva", "Produção", "2024-03-01");
